# Student Marks Prediction

## Overview

This project predicts student marks using Machine Learning based on various academic factors. The dataset is cleaned, preprocessed, and used to train a prediction model that estimates a student's final marks.

The objective is to understand how different factors affect student performance and build a model that can accurately predict marks.

---

## Features

- Cleaned and preprocessed student dataset
- Performed Exploratory Data Analysis (EDA)
- Trained a Machine Learning model
- Predicted student marks
- Evaluated model performance using suitable metrics

---

## Workflow

1. Load the student dataset.
2. Clean and preprocess the data.
3. Perform exploratory data analysis.
4. Split the data into training and testing sets.
5. Train the Machine Learning model.
6. Evaluate the model's performance.
7. Predict student marks.

---

Some possible insights include:

## Installation

Clone the repository:

```bash
git clone https://github.com/Varshini0412/MarksPrediction.git
```

Move into the project folder:

```bash
cd MarksPrediction
```

Install the required packages:

```bash
pip install -r requirements.txt
```

---

## Run the Project

Launch Jupyter Notebook:

```bash
jupyter notebook
```

Open the notebook and run the cells sequentially.
